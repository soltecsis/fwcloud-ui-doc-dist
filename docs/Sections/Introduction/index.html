# Introduction

FWCloud-UI is the user interface that allows us to manage all our firewall clouds. It is a web interface accessible through browsers such as Chrome or Firefox, developed using the Angular framework thanks to which it has a desktop application behavior, even if it's ran within our web browser.

It is a fairly simple and intuitive application to manage for users accustomed to working with firewalls based on IPTables.

This document is the user's manual for FWCloud-UI and in it we will give a detailed description of how use the FWCloud User Interface and all the features of this great web application.

Throughout the document we will give references to several Video Tutorials that will help you to understand how to use FWCloud and the most advisable use practices. These videos are part of our YouTube channel , which we update as we add new features to FWCloud-UI.

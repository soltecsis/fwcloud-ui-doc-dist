# Login

Opening your FWCloud access URL in a web browser like Chrome of Firefox you will be able to download the user interface and execute it in your browser. Once downloading is complete, the login window will be presented.

Login in for the first time

In a fresh new installation the default access data will be:

Customer code: 1
User name: fwcadmin
Password: fwcadmin


Once you are logged in, from the next screen


you can review the Configuration Options of FWCloud-UI from the green settings icon in the top righ corner, visit FWCloud-UI help


, log out from FWCloud-UI, Import a cloud, create a new cloud, access the context menu of the clouds and go into to the firewall section of a cloud.
